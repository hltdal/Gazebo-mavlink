1. MODE Değişikliği ile Durdurma (STABILIZE veya ALT_HOLD Moduna Geçiş)
Drone'un uçuş modunu acil durumlarda değiştirmek, en hızlı ve etkili yöntemlerden biridir. Eğer drone belirli bir komutu yerine getiriyorsa (takeoff, waypoint uçuşu vb.), STABILIZE veya ALT_HOLD moduna geçerek drone'un motorlarının hızını ve pozisyonunu stabilize edebilirsiniz.

Örneğin, STABILIZE moduna geçiş komutunu şu şekilde verebilirsiniz:

python
Kodu kopyala
def emergency_stop(self, drone):
    # STABILIZE moduna geçiş yaparak drone'u durdur
    drone.mav.set_mode_send(
        drone.target_system,
        mavutil.mavlink.MAV_MODE_FLAG_CUSTOM_MODE_ENABLED,
        drone.mode_mapping()['STABILIZE']
    )
    print("Acil durdurma komutu gönderildi: STABILIZE moduna geçiliyor.")
Bu komut drone'u stabil moduna geçirir ve motorları durdurmaz, ancak drone'u hareket ettirmeyi bırakır ve stabil pozisyonda tutar.

2. BRAKE Modunu Kullanma
BRAKE modu, drone'un tüm hızını aniden durdurmak için kullanılır. Bu mod, drone'u mevcut konumunda durdurmak için idealdir.

python
Kodu kopyala
def emergency_brake(self, drone):
    # BRAKE moduna geçiş yaparak drone'u acilen durdur
    drone.mav.set_mode_send(
        drone.target_system,
        mavutil.mavlink.MAV_MODE_FLAG_CUSTOM_MODE_ENABLED,
        drone.mode_mapping()['BRAKE']
    )
    print("Acil durdurma komutu gönderildi: BRAKE moduna geçiliyor.")
BRAKE modu, drone'un hızını hızlıca sıfırlayarak konumunda kalmasını sağlar.

3. LAND Komutu ile Acil İniş
Acil bir durumda drone'u tamamen durdurmak yerine doğrudan iniş yaptırmak istiyorsanız, LAND komutunu kullanabilirsiniz. Bu komut, drone'un olduğu yerden güvenli bir iniş yapmasını sağlar.

python
Kodu kopyala
def emergency_land(self, drone):
    # LAND komutu göndererek drone'u acilen indirin
    drone.mav.command_long_send(
        drone.target_system,
        drone.target_component,
        mavutil.mavlink.MAV_CMD_NAV_LAND,
        0,  # Confirmation
        0, 0, 0,  # Parametreler (Unused)
        0, 0, 0,  # Yaw angle ve enlem/boylam (Unused)
        0  # Yükseklik (Unused)
    )
    print("Acil iniş komutu gönderildi.")
4. Motorları Kapatma (DISARM)
Daha tehlikeli ama kesin bir acil durdurma yöntemi, drone'un motorlarını kapatmaktır (DISARM). Bu işlem uçuş sırasında yapılırsa, drone kontrolsüz bir şekilde yere düşebilir. Ancak aşırı acil durumlarda kullanabilirsiniz.

python
Kodu kopyala
def emergency_disarm(self, drone):
    # Drone'u DISARM yaparak motorları kapat
    drone.mav.command_long_send(
        drone.target_system,
        drone.target_component,
        mavutil.mavlink.MAV_CMD_COMPONENT_ARM_DISARM,
        0,  # Confirmation
        0,  # Disarm (1 -> Arm, 0 -> Disarm)
        0, 0, 0, 0, 0, 0
    )
    print("Acil motor kapatma komutu gönderildi: DISARM.")
Dikkat: DISARM komutu uçuş sırasında tehlikeli olabilir, bu yüzden yalnızca çok kritik durumlarda kullanılmalıdır.

Sonuç
STABILIZE veya BRAKE modları, drone'u acil bir durumda güvenli bir şekilde durdurmak için en uygun yöntemlerdir.
LAND komutu ise güvenli bir iniş sağlar.
DISARM komutu en hızlı acil durdurma yöntemidir, ancak sadece çok tehlikeli durumlarda kullanılmalıdır, çünkü drone'u kontrolsüz bir şekilde yere indirir.
Bu komutları kullanarak, drone'un takeoff işlemi sırasında veya herhangi bir uçuş görevi esnasında acil bir durumda müdahale edebilir ve güvenli bir şekilde durdurabilirsiniz.
